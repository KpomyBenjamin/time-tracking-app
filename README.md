# Time Tracking App (Flutter)

Application mobile de suivi du temps développée avec Flutter permettant de gérer efficacement ses projets et tâches.

## Fonctionnalités
- Ajout et suppression d’entrées de temps
- Gestion des projets et des tâches
- Regroupement des entrées par projet
- Stockage local des données
- Interface dynamique avec Provider

## Technologies utilisées
- Flutter
- Dart
- Provider
- Local Storage
